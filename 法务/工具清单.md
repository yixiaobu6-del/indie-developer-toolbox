## 四、合同审核清单（快速版）

### 必查6项

| 条款 | 检查要点 | 风险信号 |
|------|----------|----------|
| 服务范围 | 交付物和时间节点是否明确 | "甲方满意为止"模糊表述 |
| 付款条件 | 付款节点与交付是否对应 | 预付>50%风险大 |
| 知识产权 | 成果归属是否清晰 | 默认归甲方，需谈判 |
| 保密协议 | 期限和范围是否合理 | "永久保密"不现实 |
| 违约责任 | 罚则是否对等 | 单向高额罚则 |
| 终止条款 | 有退出路径吗 | 没有解约条款 |

### 遇到以下条款立即拒绝/修改：

1. "永久保密" → 改为"保密期2年"
2. "甲方满意为止" → 改为"按合同约定验收标准"
3. 单向高额违约金 → 要求对等
4. 知识产权默认归甲方 → 谈判争取
