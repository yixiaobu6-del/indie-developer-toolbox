## 工具选择决策框架

### 按阶段推荐组合

| 阶段 | 推荐组合 | 月成本 |
|------|----------|--------|
| **创意验证** | Google Trends + Figma + Google Forms | $0 |
| **MVP开发** | Vite + Supabase + Vercel + Prisma | $0-25 |
| **上线运营** | Sentry免费 + Logflare + Playwright | $0 |
| **增长阶段** | Linear + Stripe + Resend + Next.js | $30-50 |

### 关键的"不要做"

1. **不要一开始就买付费工具** — 先用免费版验证需求
2. **不要同时学多个新工具** — 一次只学一个，先上手再看教程
3. **不要盲目跟风** — 别人用Supabase不代表你也必须用
4. **不要在MVP阶段纠结工具选择** — 能跑通就行，后面再换
